Understanding the Box Model

Every element in web design is a rectangular box. The CSS box model describes how these boxes are sized and spaced.

Margin

Creates space around elements, outside of any defined borders. Margins are transparent.

Margin example

Border

A border that goes around the padding and content. You can style it in many ways.

Border example

Padding

Creates space around an element's content, inside of any defined borders.

Padding example

Putting It All Together

When you set the width and height properties of an element with CSS, you are setting the content area dimensions. To get the total size, you must add padding, borders and margins.
